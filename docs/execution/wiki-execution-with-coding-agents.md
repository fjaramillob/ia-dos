# Ejecutar la LLM Wiki con coding agents

La LLM Wiki es memoria durable en archivos Markdown y Git. Por eso, su creación y mantenimiento material deben realizarse normalmente mediante un **coding agent con acceso controlado al workspace o al repositorio**, no mediante un asistente conversacional que solo puede razonar sobre el proyecto.
