# Preguntas de validación

Antes de publicar `v0.1.0-alpha.1`, la fundación debe responder:

- ¿Una persona no experta entiende qué es IA-DOS?
- ¿Se distingue con claridad el framework del proyecto adoptante?
- ¿Se entiende que ChatGPT, Gemini u otro asistente puede actuar como Project Orchestrator?
- ¿La conversación externa se convierte en un input estructurado para desarrollo?
- ¿La estructura de Conversation Spaces evita mezclar dirección, producto, arquitectura y ejecución?
- ¿La LLM Wiki queda reconocida como memoria durable y fuente contextual?
- ¿La estructura app/wiki es firme sin ser obligatoria?
- ¿Está claro cuándo clonar IA-DOS y cuándo basta consumirlo por referencia?
- ¿Los handoffs hacia coding agents incluyen contexto mínimo, límites y verificación?
- ¿La documentación evita promesas futuras?
- ¿Cada tipo de información tiene una fuente de verdad?
- ¿Los términos técnicos están explicados?
- ¿El framework reduce complejidad y consumo de contexto en vez de agregarlos?

Estas preguntas se utilizarán durante la revisión de la versión alpha.
