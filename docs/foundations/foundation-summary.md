# Resumen de la fundación

IA-DOS es una capa operativa común para organizar proyectos de software desarrollados con asistencia de inteligencia artificial.

La fundación establece:

- qué problema resuelve;
- para quién está pensado;
- qué principios lo limitan;
- cómo se organiza el trabajo;
- cómo se adopta por proyecto;
- dónde vive cada tipo de información;
- cómo se gobierna el proyecto abierto.

La siguiente etapa es convertir estos fundamentos en guías prácticas de instalación, clonación, adopción y ejecución.
