# Niveles de recomendación

IA-DOS utiliza tres niveles para evitar que todas las prácticas parezcan obligatorias.

## Requerido

Condición mínima para declarar adopción de una versión de IA-DOS.

## Recomendado

Práctica que normalmente aporta valor, pero puede reemplazarse mediante una excepción documentada.

## Opcional

Práctica útil solo para determinados proyectos, equipos o niveles de riesgo.

Cada estándar futuro debe indicar su nivel.
