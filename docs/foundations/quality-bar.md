# Umbral de calidad

Un documento fundacional puede considerarse listo para revisión cuando:

- tiene un propósito claro;
- no contradice otros documentos;
- utiliza ejemplos genéricos;
- explica términos técnicos;
- distingue requisitos, recomendaciones y opciones;
- no promete capacidades inexistentes;
- evita duplicación;
- incluye límites o excepciones cuando corresponde;
- puede ser entendido por el público objetivo.

Este umbral no reemplaza la revisión humana.
