# Decisiones

Esta carpeta conserva decisiones durables del proyecto.

Cada decisión debe incluir:

- título;
- fecha;
- estado;
- contexto;
- decisión;
- alternativas consideradas;
- consecuencias;
- fuentes relacionadas.

Estados sugeridos:

```text
Propuesta
Aceptada
Reemplazada
Deprecada
```

No registres aquí conversaciones exploratorias sin una decisión confirmada.
